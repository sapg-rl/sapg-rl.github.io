PPO with other other exploration methods

We test different methods for exploration such as RND and DIAYN-based rewards, however we observed that these methods obtain similar subpar performance as is obtained through simple PPO.

Note that, to the best of our knowledge, DIAYN-based rewards are generally used to learn different skills and not maximize reward in a particular environment. To use DIAYN in our setting, we use it as an additive intrinsic reward trying to generate diversity in the policies, although we did not find an prior work that does so.

Utilizing the complete off-policy data

A way to use the complete data for off-policy updates instead of subsampling would be increase the number of mini epochs in an epochs and then use on-policy data combined with different off-policy data in each miniepochs. By appropriately setting the number of mini-epochs, we can use the complete off-policy data. We present the results in the figure below. We find that using the complete data does not lead to any additional performance gains.
